# NeuroLogic A*esque Decoding

Extends NeuroLogic Decoding with lookahead heuristics inspired by A* search. Estimates future constraint satisfaction using a look-ahead function, enabling the decoder to make globally better choices at each step rather than relying on greedy or beam-local decisions. Handles more complex constraint types including ordering and negation.
