# ReLU Family

Rectified Linear Unit and its variants — the dominant activation functions in modern deep learning.

## Sub-directories

| Directory | Description |
|-----------|-------------|
| [ReLU](ReLU/) | f(x) = max(0, x) — the standard |
| [Leaky_ReLU](Leaky_ReLU/) | f(x) = max(αx, x), small slope for negatives |
| [PReLU](PReLU/) | Parametric ReLU with learnable slope |
| [ELU](ELU/) | Exponential Linear Unit |
| [SELU](SELU/) | Scaled ELU for self-normalizing networks |
