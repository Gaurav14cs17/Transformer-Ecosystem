# ZeRO (Zero Redundancy Optimizer)
Eliminates memory redundancy in data parallelism by partitioning optimizer states (Stage 1), gradients (Stage 2), and parameters (Stage 3) across GPUs. Achieves model-parallel memory efficiency with data-parallel computational efficiency.
