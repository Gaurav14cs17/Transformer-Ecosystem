# GELU (Gaussian Error Linear Unit)

f(x) = x · Φ(x), where Φ is the standard Gaussian CDF

Default activation in BERT, GPT-2, GPT-3, and many modern transformers. Smooth approximation to ReLU with stochastic regularization properties.
