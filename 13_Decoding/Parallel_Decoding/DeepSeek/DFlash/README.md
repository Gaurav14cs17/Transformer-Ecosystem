# DFlash

Applies block diffusion to speculative decoding, enabling flash-speed parallel token generation. Instead of autoregressively drafting tokens, DFlash uses a diffusion-based process to generate entire blocks of draft tokens in parallel, achieving up to 6x speedup over standard autoregressive decoding.
