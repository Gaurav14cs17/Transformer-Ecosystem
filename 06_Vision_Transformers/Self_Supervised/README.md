# Self-Supervised Vision Transformers
Pre-training methods for ViT without labeled data.
## Sub-directories
| [MAE](MAE/) | Masked Autoencoder |
| [DINO](DINO/) | Self-distillation without labels (v1, v2) |
| [iBOT](iBOT/) | Image BERT with online tokenizer |
| [SimMIM](SimMIM/) | Simple masked image modeling |
| [I_JEPA](I_JEPA/) | Image Joint Embedding Predictive Architecture |
| [V_JEPA](V_JEPA/) | Video Joint Embedding Predictive Architecture |
