# LUCID Attention

Attention with Preconditioned Representations. Applies a preconditioner derived from key-key similarities to minimize overlap between keys, allowing precise retrieval among large numbers of keys.

Achieves up to 18% improvement on BABILong and 14% on RULER multi-needle retrieval.
