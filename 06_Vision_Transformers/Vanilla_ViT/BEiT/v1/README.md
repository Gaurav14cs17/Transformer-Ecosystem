# BEiT — BERT Pre-Training of Image Transformers

## Overview

BEiT (Bidirectional Encoder representation from Image Transformers) adapts the BERT-style masked language modeling paradigm to vision. It pre-trains Vision Transformers by masking random image patches and predicting their discrete visual tokens, generated by a pre-trained discrete variational autoencoder (dVAE). This self-supervised approach achieves strong performance on image classification and semantic segmentation.

## Architecture & Mechanism

BEiT uses a two-stage approach:

1. **Image tokenization**: A pre-trained dVAE (from DALL-E) encodes the image into a grid of discrete visual tokens (e.g., 14×14 tokens from a 8192-entry codebook).
2. **Masked image modeling (MIM)**: Random patches (typically 40%) of the input image are masked, and the ViT is trained to predict the corresponding visual tokens.
   - Input: Masked image patches → ViT encoder → prediction head.
   - Target: Discrete visual tokens from the dVAE (not raw pixel values).
   - Loss: Cross-entropy over the visual token vocabulary.

## Key Innovations

- **Visual token prediction**: Predicting discrete tokens rather than raw pixels provides a higher-level reconstruction target, similar to predicting word tokens in BERT.
- **Blockwise masking**: Masks contiguous blocks of patches (rather than random individual patches) to create more challenging predictions.
- **Self-supervised pre-training**: No labels needed during pre-training; learns strong visual representations from unlabeled data.
- **Transfer to dense tasks**: Pre-trained representations transfer well to semantic segmentation (ADE20K) in addition to classification.
