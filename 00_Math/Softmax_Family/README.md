# Softmax Family

Softmax normalization and its variants — converting logits to probability distributions.

## Sub-directories

| Directory | Description |
|-----------|-------------|
| [Standard_Softmax](Standard_Softmax/) | Classical softmax normalization |
| [Sparse_Softmax](Sparse_Softmax/) | Sparsemax and α-entmax |
| [Log_Softmax](Log_Softmax/) | Numerically stable log-softmax |
| [Safe_Softmax](Safe_Softmax/) | Overflow/underflow-safe implementations |
| [QA_Softmax](QA_Softmax/) | Query-aware softmax variants |
