# Sigmoid (Logistic Function)

The classical sigmoid activation function: σ(x) = 1/(1+e⁻ˣ)

Introduced in 1845, it maps any real number to (0, 1). Foundation of logistic regression and historically the default activation for neural networks before ReLU.
