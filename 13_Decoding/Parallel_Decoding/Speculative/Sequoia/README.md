# Sequoia

A hardware-aware speculative decoding framework that constructs optimal token trees adapted to the specific hardware configuration (GPU memory bandwidth, compute). Uses a dynamic programming algorithm to determine the best tree topology given a draft model's token probabilities and the target hardware's verification throughput constraints.
