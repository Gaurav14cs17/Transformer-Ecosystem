# FSDP (Fully Sharded Data Parallel)
PyTorch's native implementation of ZeRO-3 style sharding. Shards model parameters, gradients, and optimizer states across data-parallel workers. Supports mixed precision, activation checkpointing, and CPU offloading out of the box.
