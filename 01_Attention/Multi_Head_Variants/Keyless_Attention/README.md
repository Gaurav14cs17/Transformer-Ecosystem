# Keyless Attention

Eliminates the key projection entirely, operating over queries and values only. Yields a Value-Only Cache that reduces KV cache memory by exactly 50%.

Introduces Depth-m Attention Factorization — a generalization where standard attention is depth-2 and Keyless is depth-m.
