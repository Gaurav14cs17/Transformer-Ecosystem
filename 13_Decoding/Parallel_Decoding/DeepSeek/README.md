# DeepSeek Speculative Decoding Family

DeepSeek's family of speculative decoding methods that accelerate autoregressive inference through diffusion-based drafting and confidence-adaptive parallelism.

## Sub-directories
| Directory | Description |
|-----------|-------------|
| [DFlash](DFlash/) | Block diffusion for flash speculative decoding (6x speedup) |
| [DSpark](DSpark/) | Confidence-scheduled semi-autoregressive decoding |
