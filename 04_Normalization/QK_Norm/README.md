# QK-Norm (Query-Key Normalization)
Normalizes Q and K vectors before dot-product to prevent attention logit explosion. Used in Gemma 2/3, OLMo 2, Qwen 3.
## Sub-directories
| [Lp_QKNorm](Lp_QKNorm/) | Enhanced QKNorm with Lp norm |
| [QK_Normed_MLA](QK_Normed_MLA/) | QK Normalization for MLA without full key caching |
