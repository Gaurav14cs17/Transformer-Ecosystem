# EAGLE (Extrapolation Algorithm for Greater Language-model Efficiency)
Performs autoregression at the feature level rather than the token level. A lightweight draft model operates on the second-to-last layer features of the target model, enabling more accurate multi-token speculation with tree-structured verification.
