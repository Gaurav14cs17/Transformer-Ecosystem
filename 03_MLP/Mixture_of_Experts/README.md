# Mixture of Experts (MoE)
Conditional computation where each token is routed to a subset of expert FFN networks.
## Sub-directories
| [Sparsely_Gated_MoE](Sparsely_Gated_MoE/) | Original sparsely-gated MoE |
| [GShard](GShard/) | Scalable MoE with automatic sharding |
| [Switch_Transformer](Switch_Transformer/) | Single-expert routing |
| [Expert_Choice](Expert_Choice/) | Experts choose tokens (not vice versa) |
| [Soft_MoE](Soft_MoE/) | Soft routing without discrete selection |
| [Mixtral](Mixtral/) | Mistral's MoE architecture |
| [DeepSeek_MoE](DeepSeek_MoE/) | Fine-grained expert segmentation |
| [DirMoE](DirMoE/) | Dirichlet-routed MoE (ICLR 2026) |
| [PathMoE](PathMoE/) | Path-constrained MoE |
| [ProbMoE](ProbMoE/) | Probabilistic routing MoE |
