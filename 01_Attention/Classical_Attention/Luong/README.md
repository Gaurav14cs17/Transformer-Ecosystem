# Luong Attention

Multiplicative (dot-product) attention that computes alignment scores via direct dot product or general bilinear form.

score(s_t, h_i) = s_t^T · h_i (dot) or s_t^T · W · h_i (general)

Simpler and faster than Bahdanau attention, with comparable performance.
