# Embeddings

Embedding layers that convert discrete or continuous inputs into dense vector representations.

## Sub-directories

| Directory | Description |
|-----------|-------------|
| [Token_Embedding](Token_Embedding/) | Learned lookup table for discrete tokens |
| [Patch_Embedding](Patch_Embedding/) | Image patch to embedding (ViT-style) |
| [Word_Embedding](Word_Embedding/) | Word2Vec, GloVe, FastText |
| [Rotary_Embedding](Rotary_Embedding/) | RoPE — position via rotation matrices |
