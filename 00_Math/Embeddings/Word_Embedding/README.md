# Word Embedding

Pre-trained dense vector representations of words capturing semantic relationships.

## Methods
- **Word2Vec**: Skip-gram and CBOW models (Mikolov et al., 2013)
- **GloVe**: Global vectors from co-occurrence statistics (Pennington et al., 2014)
- **FastText**: Subword embeddings for morphologically rich languages (Bojanowski et al., 2017)

## Key Properties
- Captures semantic similarity: king - man + woman ≈ queen
- Pre-transformer era foundation for NLP
- Now largely superseded by contextual embeddings from transformers
