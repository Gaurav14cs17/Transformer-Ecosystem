# NeuroLogic Decoding

Integrates predicate logic constraints into neural text generation. Allows specifying hard constraints (e.g., "output must contain these keywords") as conjunctive normal form (CNF) clauses, then uses a beam-based decoding algorithm that prunes candidates violating the constraints while maintaining generation quality.
