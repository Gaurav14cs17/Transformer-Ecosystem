# DSpark

Confidence-scheduled speculative decoding with semi-autoregressive generation. Dynamically adjusts the degree of parallelism at each step based on the model's confidence: when confident, generates more tokens in parallel; when uncertain, falls back toward sequential decoding for accuracy.
