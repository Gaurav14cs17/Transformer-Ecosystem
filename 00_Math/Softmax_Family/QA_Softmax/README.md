# QA Softmax (Query-Aware Softmax)

Query-aware softmax variants that condition the normalization on the query vector, allowing different queries to produce different attention temperature or sharpness.

## Key Properties
- Temperature depends on query content
- Adaptive sharpness per query head
- Used in modern attention variants like Affine-Scaled Attention
