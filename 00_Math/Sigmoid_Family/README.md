# Sigmoid Family

Activation functions based on the sigmoid (logistic) curve and its variants.

## Sub-directories

| Directory | Description |
|-----------|-------------|
| [Sigmoid](Sigmoid/) | Classical logistic sigmoid σ(x) = 1/(1+e⁻ˣ) |
| [Swish](Swish/) | Self-gated activation: x · σ(βx) |
| [Mish](Mish/) | Self-regularized non-monotonic: x · tanh(softplus(x)) |
