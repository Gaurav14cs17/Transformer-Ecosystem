# Long-Context Attention

Mechanisms designed to extend transformer context windows beyond standard limits.

## Sub-directories

| Directory | Description |
|-----------|-------------|
| [Transformer_XL](Transformer_XL/) | Segment-level recurrence with relative PE |
| [Longformer](Longformer/) | Sliding window + global attention |
| [BigBird](BigBird/) | Random + window + global sparse attention |
| [Memorizing_Transformer](Memorizing_Transformer/) | kNN retrieval over past activations |
| [Infini_Attention](Infini_Attention/) | Compressive memory for infinite context |
| [LM_Infinite](LM_Infinite/) | Length generalization without fine-tuning |
