# Multi-Latent Attention (MLA)

Compresses KV cache into low-rank latent vectors. Caches only d_c-dimensional latent states instead of full keys and values, achieving significant memory reduction.

Introduced in DeepSeek-V2 and used in DeepSeek-V3/V4.
