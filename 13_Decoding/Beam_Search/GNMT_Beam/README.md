# GNMT Beam Search

Google's Neural Machine Translation beam search with length normalization and coverage penalty. Addresses the bias of standard beam search toward shorter sequences by normalizing scores by sequence length, and adds a coverage penalty to encourage the model to attend to all parts of the source sentence.
