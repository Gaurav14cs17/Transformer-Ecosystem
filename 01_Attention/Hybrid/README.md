# Hybrid Attention

Architectures that combine attention with recurrence, convolution, or state-space models.

## Sub-directories

| Directory | Description |
|-----------|-------------|
| [RetNet](RetNet/) | Retentive Network — multi-scale exponential decay |
| [RWKV](RWKV/) | Receptance Weighted Key Value (v5, Eagle/Finch) |
| [Hyena](Hyena/) | Long convolution-based attention alternative |
| [Jamba](Jamba/) | Hybrid Transformer-Mamba architecture |
| [Mamba_Attention](Mamba_Attention/) | Combined Mamba + attention layers |
